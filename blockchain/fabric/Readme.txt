Those 2 folders "myapp"and "mychaincode" are meant to be put inside the fabric-samples folder that you can get by downloading: curl -sSLO https://raw.githubusercontent.com/hyperledger/fabric/main/scripts/install-fabric.sh && chmod +x install-fabric.sh

more info at: https://hyperledger-fabric.readthedocs.io/en/latest/install.html

Also it can be needed the command: "npm install" 
inside the "myapp" and "mychaincode" folders
